# powercycle
A command-line util to manage powercycle.

>  ## Usage: 
>   `powercycle [hh:mm:ss | now | timeleft(in minutes)]`
> 
> ## Building from source:
> 1. call `cmake`
> 2. call make
> 2. you are good to go ! refer Usage
